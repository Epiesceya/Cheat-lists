git config - настройка гит (system < global < local)

Анализ изменений и истории:
git branch - показать список веток
git tag - показать список тегов(версий)
git status - посмотреть состояния файлов в индексе
git log(shortlog) - посмотреть историю коммитов
git reflog - посмотреть локальную историю коммитов(можно найти удаленные коммиты сроком до 90 дней)
git show <commit_id> - посмотреть изменения коммита
git diff - посмотреть разницу между изменениями


git init - создать локальный репозиторий(на PC)
git clone - клонировать локальный репозиторий
git add - добавить файл к отслеживанию в индекс
git commit - сделать коммит(лучше через -m)
git push - отправить на внешний репозиторий(GitHub)
git pull - синхронизировать внешний и локальный репозитории
git reset - исключить файл из индекса(HEAD~1 - на шаг назад; <commit_id> - к определенному коммиту; ORIG_HEAD - к предыдущему коммиту)
git revert - создает новый коммит, который отменяет изменения указанного коммита, не удаляя его из истории (--abort - отменить незавершенный откат коммита)
git rm file.format - удалить файл и зафиксировать в индекс
git clean - удалить Untracked файлы (-f,-d,-n)
git reflog expire - очистить историю коммитов
git gc - очистить историю коммитов
git mv - переименовать\переместить файл и зафиксировать в индекс
git diff - показать изменения после предыдущего коммита

Отправить изменения на внешний репозиторий:
git add .; git commit -m "Описали что сделали"; git push

. - текущая директория

Игнорировать и не отслеживать в репозитории:
nano .gitignore
в файле:
filename.format - конкретный файл
*.format - все файлы определенного формата
folder/* - все файлы в папке
!filename.format - исключение
git config --global core.excludesfile ~/.gitignore_global - глобальная настройка для пути к файлу .gitignore

 Флаги:

-a - пропустить git add(кроме Untracked файлов)
-m - добавить сообщение
-am - добавить файл к коммиту и сообщение
--amend - исправить сообщение к коммиту ДО пуша
-am - добавить файл к коммиту и сообщение(кроме Untracked файлов)
--abort - 
Для git log:
--oneline - в одну строку
--all - все ветки и теги
--p - разница между коммитами
--author=name - по автору
--since=date - после указанной даты
--until=date - до указанной даты
--file_name - по имени файла
--grep=pattern - по слову(pattern) в сообщении к коммиту
Для git reset:
--soft - отменяет коммит
--mixed - отменяет коммит и индексацию(по умолчанию)
--hard - отменяет коммит, индексацию и все изменения в файлах

Статус файлов:
Untracked - не отслеживается в индексе 
Staged - добавлен в индекс
Unmodified - закоммичен
Modified - изменен, но не отслеживается
